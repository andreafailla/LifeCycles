

Welcome to LifeCycles' documentation!
=======================================

``lifecycles`` is a Python software package designed to track and analyze the longitudinal evolution of groups of items,
e.g., data clusters, graph communities, etc.


If you would like to test ``lifecycles`` functionalities without installing it on your machine consider using the preconfigured Jupyter Hub instances offered by the H2020 `SoBigData++`_ research project.
